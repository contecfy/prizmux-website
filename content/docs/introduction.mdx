# Introduction to Prizmux

Prizmux is a **developer-first React Native component system** designed for production applications. It provides a collection of battle-tested, fully-typed components that focus on **zero unnecessary dependencies** and maximum flexibility.

## Why Prizmux?

Building production-grade React Native applications often means dealing with complex component libraries that bring along heavy dependencies and inflexible designs. Prizmux takes a different approach:

### Zero Unnecessary Dependencies
Every component in Prizmux is built with minimal dependencies. We believe in shipping less JavaScript, which means faster apps and fewer version conflicts.

### Fully Typed
Complete TypeScript support means better IDE autocomplete, fewer runtime errors, and improved developer experience. All props are fully documented.

### Bring Your Own Icons
Don't like our icon choice? No problem. Prizmux lets you bring your own icon library or use whatever works best for your design system.

### Production Ready
Used in production by developers building real applications. Components handle edge cases, support accessibility, and work across different React Native platforms.

## Component Library

Prizmux includes a collection of essential components:

- **Button** — Flexible button component with variants and sizes
- **Card** — Composable card for content grouping
- **Bottom Sheet** — Smooth bottom sheet modal
- **Image Preview** — Image gallery with touch gestures
- **Header with Back** — Navigation header with back button
- **Empty State** — Placeholder for empty data states
- **Phone Input** — Formatted phone number input

## Philosophy

We believe component libraries should:

1. **Be lightweight** — Dependencies add bloat and maintenance burden
2. **Be flexible** — Adapt to your design, not the other way around
3. **Be explicit** — Clear, predictable APIs without hidden magic
4. **Be accessible** — Support screen readers and keyboard navigation
5. **Be typed** — TypeScript support out of the box

## Who Should Use Prizmux?

Prizmux is perfect for:
- Production React Native applications
- Teams building design systems
- Projects that value minimal dependencies
- Developers who want full control over their components
- Applications targeting multiple platforms (iOS, Android, Web)
