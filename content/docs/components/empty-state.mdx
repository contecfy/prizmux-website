# Empty State

EmptyState is a component that displays helpful messaging when there's no data to show. Perfect for guiding users when content is unavailable.

## Overview

EmptyState combines an icon, title, description, and optional action to create a complete empty state experience. It helps users understand why nothing is displayed and what to do next.

**Use it for:**
- No search results
- Empty lists or feeds
- First-time user experiences
- Data not available states
- Error fallbacks

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon` | `React.ReactNode` | - | Icon to display |
| `title` | `string` | - | Main message |
| `description` | `string` | - | Secondary message |
| `action` | `React.ReactNode` | - | Action button |
| `onAction` | `() => void` | - | Action callback |
| `variant` | `'default' \| 'compact'` | `'default'` | Layout variant |

## Usage

### Basic Empty State

```tsx
import { EmptyState } from 'prizmux'
import { Search } from 'lucide-react-native'

export default function SearchResults() {
  return (
    <EmptyState
      icon={<Search size={48} />}
      title="No Results"
      description="Try adjusting your search terms"
    />
  )
}
```

### With Action

```tsx
import { EmptyState, Button } from 'prizmux'
import { Plus } from 'lucide-react-native'

<EmptyState
  icon={<Plus size={48} />}
  title="No Items"
  description="Create your first item to get started"
  action={
    <Button 
      title="Create Item"
      onPress={() => navigation.navigate('Create')}
    />
  }
/>
```

### No Search Results

```tsx
import { EmptyState } from 'prizmux'
import { SearchX } from 'lucide-react-native'

function SearchList() {
  const [results, setResults] = useState([])
  const [query, setQuery] = useState('')

  return (
    <>
      <SearchBar value={query} onChange={setQuery} />
      {results.length === 0 ? (
        <EmptyState
          icon={<SearchX size={48} />}
          title="No Results"
          description={`No items found for "${query}"`}
        />
      ) : (
        <List data={results} />
      )}
    </>
  )
}
```

### Empty Feed

```tsx
  }

```

### Error State

```tsx
import { EmptyState, Button } from 'prizmux'
import { AlertCircle } from 'lucide-react-native'

function LoadingScreen() {
  const [error, setError] = useState(null)

  return error ? (
    <EmptyState
      icon={<AlertCircle size={48} />}
      title="Something Went Wrong"
      description={error.message}
      action={
        <Button 
          title="Try Again"
          onPress={() => retryLoad()}
        />
      }
    />
  ) : (
    <LoadingList />
  )
}
```

## Notes

- Use meaningful icons that relate to the empty state
- Keep titles concise and action-oriented
- Descriptions can provide helpful context
- Action button is optional but recommended
- Automatically scales on different screen sizes
- Works in both light and dark modes
