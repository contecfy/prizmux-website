A navigation header component with back button, title, optional avatar, and customizable action buttons. Perfect for detail screens and navigation flows.

## Overview

HeaderWithBack simplifies common header patterns while remaining flexible for custom needs. It handles layout, spacing, and interactions automatically.

**Use it for:**
- Screen headers and navigation
- Detail screens
- Profile screens
- Modal headers
- Settings screens
- Chat screens

## Basic Usage

```tsx
import { HeaderWithBack } from 'prizmux'
import { View, Text } from 'react-native'

export default function DetailsScreen({ navigation }) {
  return (
    <>
      <HeaderWithBack
        title="Product Details"
        onBackPress={() => navigation.goBack()}
      />
      <View style={{ flex: 1, padding: 16 }}>
        {/* Screen content */}
      </View>
    </>
  )
}
```

## With Avatar

```tsx
import { HeaderWithBack } from 'prizmux'
import { Image } from 'react-native'

function ChatScreen({ navigation, contact }) {
  return (
    <HeaderWithBack
      title={contact.name}
      avatar={
        <Image
          source={{ uri: contact.avatar }}
          style={{ width: 40, height: 40, borderRadius: 20 }}
        />
      }
      onBackPress={() => navigation.goBack()}
    />
  )
}
```

## With Action Buttons

```tsx
import { HeaderWithBack } from 'prizmux'
import { Text } from 'react-native'

function ArticleScreen({ navigation }) {
  return (
    <HeaderWithBack
      title="Article"
      onBackPress={() => navigation.goBack()}
      actions={[
        {
          icon: <Text style={{ fontSize: 20 }}>📤</Text>,
          onPress: () => shareArticle(),
          badge: undefined
        },
        {
          icon: <Text style={{ fontSize: 20 }}>⋮</Text>,
          onPress: () => showMenu(),
        }
      ]}
    />
  )
}
```

## With Badge Notifications

```tsx
import { HeaderWithBack } from 'prizmux'
import { Text } from 'react-native'

function MessagesScreen({ navigation, unreadCount }) {
  return (
    <HeaderWithBack
      title="Messages"
      onBackPress={() => navigation.goBack()}
      actions={[
        {
          icon: <Text style={{ fontSize: 20 }}>🔔</Text>,
          onPress: () => openNotifications(),
          badge: unreadCount > 0 ? unreadCount : undefined
        }
      ]}
    />
  )
}
```

## Title Positioning

```tsx
{/* Left-aligned title */}
<HeaderWithBack
  title="Edit Profile"
  titlePosition="left"
  onBackPress={() => navigation.goBack()}
/>

{/* Center-aligned title (default) */}
<HeaderWithBack
  title="Details"
  titlePosition="center"
  onBackPress={() => navigation.goBack()}
/>

{/* Right-aligned title */}
<HeaderWithBack
  title="Info"
  titlePosition="right"
  onBackPress={() => navigation.goBack()}
/>
```

## Props Reference

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | **Required** | Header title text |
| `onBackPress` | `() => void` | **Required** | Back button callback |
| `avatar` | `ReactNode` | - | Avatar component (renders as-is) |
| `titlePosition` | `'left' \| 'center' \| 'right'` | `'center'` | Title alignment |
| `backIcon` | `ReactNode` | - | Custom back icon (defaults to ‹) |
| `actions` | `HeaderAction[]` | - | Action buttons array |

## Usage

### Basic Header

```tsx
import { HeaderWithBack } from 'prizmux'

export default function DetailsScreen() {
  return (
    <>
      <HeaderWithBack
        title="Product Details"
        onBack={() => navigation.goBack()}
      />
      {/* Screen content */}
    </>
  )
}
```

### With Subtitle

```tsx
<HeaderWithBack
  title="Messages"
  subtitle="John Doe"
  onBack={() => navigation.goBack()}
/>
```

### With Action Button

```tsx
import { HeaderWithBack } from 'prizmux'
import { Share2 } from 'lucide-react-native'

<HeaderWithBack
  title="Article"
  onBack={() => navigation.goBack()}
  rightAction={<Share2 size={24} />}
  onRightAction={() => shareArticle()}
/>
```

### With Multiple Right Actions

```tsx
import { HeaderWithBack, Button } from 'prizmux'
import { MoreVertical } from 'lucide-react-native'
import { View } from 'react-native'

<HeaderWithBack
  title="Settings"
  onBack={() => navigation.goBack()}
  rightAction={
    <View style={{ flexDirection: 'row', gap: 8 }}>
      <Button icon={<Edit size={24} />} variant="ghost" />
      <Button icon={<MoreVertical size={24} />} variant="ghost" />
    </View>
  }
/>
```

### Custom Back Icon

```tsx

<HeaderWithBack
  title="Screen Title"
  backIcon={<ChevronLeft size={28} color="red" />}

```

### In Navigation Stack

```tsx
import { HeaderWithBack } from 'prizmux'
import { useFocusEffect } from '@react-navigation/native'

export default function ProfileScreen({ navigation }) {
  useFocusEffect(() => {
    navigation.setOptions({
      header: () => (
        <HeaderWithBack
          title="Profile"
          onBack={() => navigation.goBack()}
          rightAction={<Edit size={24} />}
          onRightAction={() => navigation.push('EditProfile')}
        />
      ),
    })
  })

  return (
    // Screen content
  )
}
```

### Modal Header

```tsx
import { HeaderWithBack } from 'prizmux'
import { X } from 'lucide-react-native'

<HeaderWithBack
  title="Filter Options"
  backIcon={<X size={24} />}
  onBack={() => dismissModal()}
/>
```

### With Loading State

```tsx
import { HeaderWithBack, Button } from 'prizmux'
import { Save } from 'lucide-react-native'
import { useState } from 'react'

export default function EditScreen() {
  const [saving, setSaving] = useState(false)

  const handleSave = async () => {
    setSaving(true)
    await saveChanges()
    setSaving(false)
  }

  return (
    <>
      <HeaderWithBack
        title="Edit Profile"
        onBack={() => navigation.goBack()}
        rightAction={
          <Button
            icon={<Save size={20} />}
            loading={saving}
            variant="ghost"
            onPress={handleSave}
          />
        }
      />
    </>
  )
}
```

## Notes

- Automatically handles safe area insets on notched devices
- Back button provides haptic feedback on press
- Right action can be any component (Button, Icon, custom)
- Title is automatically truncated if too long
- Subtitle appears below title in smaller text
- Respects system back gesture on iOS and Android
